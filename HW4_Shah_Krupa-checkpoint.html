Inc. 5000 Analysis¶

The Inc. 5000 list is a prestigious annual ranking that recognizes the fastest-growing private companies in the United States. To be considered for this list, companies must meet certain criteria. They need to demonstrate substantial revenue growth from 2014 to 2017, starting with a minimum revenue of 100,000 in 2014 and reaching at least 2 million in 2017. These companies must also be founded and generating revenue by March 31, 2014, and they must be U.S.-based, privately held, for-profit, and independent entities, not subsidiaries or divisions of other companies, as of December 31, 2017. In cases where companies have equal growth rates, those with higher revenues are ranked higher on the list. Please let me know if you need further clarification or more specific details.

#Here we will try working on a function which should return a data frame containing the top n rows of companies, based on revenues for a specific value in the category, only if both category andvalue are provided. 
def filter_and_display_top_companies(data_frame, category=None, value=None, n=10):
    if category is None and value is None:
        # If category and value are not provided, display top n companies overall
        top_n_companies = data_frame.nlargest(n, 'REVENUE')
    elif category is not None and value is not None:
        # If both category and value are provided, filter companies based on the specified category and value
        filtered_data = data_frame[data_frame[category] == value]
        top_n_companies = filtered_data.nlargest(n, 'REVENUE')
    return top_n_companies
